# AI Image Bot (на tgx)

Полноценный бот генерации картинок. Бэкенд — бесплатный Pollinations (без API-ключа),
нужен только токен бота.

## Что умеет
- Морфящаяся конфиг-карточка: модель, формат (1:1 / 16:9 / 9:16), улучшение промпта.
- Toggle-кнопки — карточка сама перерисовывает то же сообщение.
- Любой текст = промпт → реальная генерация → фото в ответ.
- Настройки и последний промпт персистентны (переживают рестарт).
- «🔁 Повторить последний».

## Запуск
1. Создай бота у [@BotFather](https://t.me/BotFather) → получи токен.
2. Положи токен в `.env` рядом с этим файлом:
   ```
   BOT_TOKEN=123456:ABC-твой-токен
   ```
3. Из корня репозитория:
   ```
   npm run build
   node --env-file=examples/ai-image-bot/.env --experimental-strip-types examples/ai-image-bot/bot.ts
   ```
4. В Telegram: `/start`, покрути параметры, пришли промпт (например `neon city at night`).
