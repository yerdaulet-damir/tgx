# tgx — примеры ботов

Реальные боты на tgx. Каждый — рабочий, компилируется, читается сверху вниз.
Все под MIT: копируй в свой проект и владей кодом.

## Как запустить любой

```bash
npm run build
# создай бота у @BotFather, положи токен в examples/<bot>/.env:  BOT_TOKEN=...
node --env-file=examples/<bot>/.env --experimental-strip-types examples/<bot>/bot.ts
```

## Витрина

| Бот | Что показывает | Ключевые примитивы |
|---|---|---|
| **ai-image-bot** | Реальная AI-генерация (Pollinations, без ключа). Панель едет вниз к результату. | `screen()`, `.mode("feed")`, `.pick/.toggle`, `.onPrompt`, `analytics` |
| **shop-bot** | Каталог с пагинацией → товар → корзина → оплата Telegram Stars. | `.view()`, корзина в state, `.onPayment`, invoice XTR |
| **subscription-bot** | Онбординг-подписка со Stars: email → план → оплата. | `flow()`, `ask/choose/payStars` |
| **habit-tracker** | Привычки со стриками, ежедневные отметки, `/stats`. | `.row()` (динамич. кнопки), персистентность, аналитика |
| **support-flow** | Саппорт КАК блок-схема: меню → категория → ветки → ввод. | `.nav()`, `c.go/back/reset`, ветвление, данные по стрелке |
| **booking-bot** | Запись на приём: календарь → тайм-пикер → подтверждение. | `ui.calendar`, `ui.calStep`, `ui.timePicker`, breadcrumb |
| **album-bot** | Обработка файлов: единый `IncomingFile`, буфер альбомов, chat-action. | `.onInput`, `.onAlbum`, `msg.file`, `c.action` |
| **ui-showcase** | Витрина UI-кита: табы, степпер, рейтинг, прогресс, confirm. | `tgx-kit/ui` |
| **movie-search** | Поиск → список с пагинацией → карточка деталей → назад. | морфящийся `.view()`, внешний API + `c.call` |
| **menu-app** | Вложенные экраны, «назад», меню. | навигационный стек `go/back` |
| **syntx-bot** | Конфигуратор генерации: бренд → модель → параметры → рефы. | морфящаяся карточка, toggle, сбор файлов |
| **image-gen** | Генерация со слотами загрузки фото (валидация типа/размера). | `card()` из `tgx-kit`, слоты |
| **tiktok-dl** | «Пришли ссылку — получи видео». | `.onInput`, `ctx.tg` (сырой grammY) |

## Что во всех работает из коробки (ноль строк от разработчика)

- **Персистентность** — состояние и позиция в навигации переживают рестарт.
- **Испарение кнопок** — старые кнопки гаснут при переходе; устаревшие — инертны.
- **Авто-answer** callback'ов, глотание «message is not modified».
- **Устойчивость** — сбой одного апдейта не роняет бота.
- **Матчинг платежа** — Stars сами привязываются к юзеру.
- **Аналитика воронки** — `analytics: "path"` + команда `/stats`.
