# Changelog

All notable changes to this project are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to
[Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- `tgx` — the engine: screens as a function of state, persistent
  navigation stack, keyboard evaporation, callback auto-answer, Telegram Stars
  payment matching, zero-code funnel analytics, file normalization, menus.
- `tgx-kit` — copy-in UI components: `card` with upload slots, keyboards,
  calendar, text builders.
- `flow` / `ask` / `choose` / `payStars` linear-wizard sugar over the engine.
- 14 example bots and a grammY side-by-side comparison.
- Smoke test suite for the engine (`npm test`).
