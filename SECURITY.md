# Security Policy

## Reporting a vulnerability

Please do not open a public issue for security vulnerabilities.

Instead, email **yerdauletdev@gmail.com** with:

- a description of the issue,
- steps to reproduce,
- the affected version.

You will get an acknowledgement within a few days. Once the issue is confirmed and
fixed, a patched release will be published and the reporter credited (unless you prefer
to stay anonymous).

## Supported versions

This project is in alpha; only the latest published version receives security fixes.
